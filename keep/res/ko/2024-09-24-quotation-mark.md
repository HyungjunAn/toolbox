# 큰 따옴표
대화, 인용, 특별 어구 따위를 나타냄

## 글 가운데서 직접 대화를 표시
```
“전기가 없었을 때는 어떻게 책을 보았을까?”
“그야 등잔불을 켜고 보았겠지.”
```

## 남의 말을 인용
```
예로부터 “민심은 천심이다.”라고 하였다.
“사람은 사회적 동물이다.”라고 말한 학자가 있다.
```

# 작은따옴표

## 따온 말 가운데 다시 따온 말이 들어 있을 때
```
“여러분! 침착해야 합니다. ‘하늘이 무너져도 솟아날 구멍이 있다.’고 합니다.”
```

## 마음 속으로 한 말
```
‘만약 내가 이런 모습으로 돌아간다면, 모두들 깜짝 놀라겠지.’
```

## 문장에서 중요한 부분 강조
```
지금 필요한 것은 ‘지식’이 아니라 ‘실천’입니다.
```
